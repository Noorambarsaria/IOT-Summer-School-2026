# Week 1

This folder contains the projects and exercises completed during the first week of learning embedded systems and Arduino programming.

## Contents

###  7 Segment Display
- Basic interfacing of a 7-segment display with Arduino.
- Displaying numbers using digital output pins.

###  4×4 LED Displaying Initials
- Displaying simple alphabet initials on a 4×4 LED matrix.
- Introduction to LED matrix control and multiplexing.

###  LED Blinking
- The classic Arduino "Blink" program.
- Understanding digital output and timing using `delay()`.

## Technologies Used
- Arduino Uno
- Arduino IDE
- C/C++
- Basic Electronic Components (LEDs, Resistors, Breadboard, Jumper Wires)

## Learning Outcomes
- Understanding Arduino digital I/O.
- Interfacing LEDs and displays.
- Writing simple embedded C programs.
- Building confidence with basic hardware connections and circuit design.
